# Financial Dashboard

A modern financial accounting system dashboard built with React, Node.js, and PostgreSQL. Provides comprehensive tools for transaction management, financial reporting, and business analytics.

## Features

- 📊 **Dashboard Overview** - Real-time financial metrics and KPIs
- 💳 **Transaction Management** - Track income and expenses
- 📈 **Financial Reports** - Generate comprehensive financial statements
- 👤 **User Authentication** - Secure login and authorization
- 🌍 **Multi-currency Support** - Handle multiple currencies
- 📱 **Responsive Design** - Works seamlessly on all devices
- 🎨 **Modern UI/UX** - Clean and intuitive interface

## Tech Stack

### Frontend
- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Redux Toolkit** - State management
- **React Router** - Navigation
- **Recharts** - Data visualization
- **Axios** - HTTP client

### Backend
- **Node.js** - Runtime
- **Express.js** - Web framework
- **PostgreSQL** - Database
- **Sequelize** - ORM
- **JWT** - Authentication
- **Joi** - Validation
